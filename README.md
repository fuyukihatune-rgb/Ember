# Ember

**やめさせない**喫煙記録アプリ。禁煙アプリはたくさんあるけれど、「ただ記録したい」喫煙者のための道具は意外とない——というところから生まれました。
責めず、急かさず、吸った一本をワンタップで記録して、自分のペースを把握します。

- ビルド不要・1ファイル完結（`index.html` + `manifest.json` + `sw.js`）
- 記録はすべて端末内（localStorage キー `ember_v1`）。広告・追跡・ログインなし。オフライン対応（PWA）
- 機能
  - ワンタップ記録／時刻・メモ指定での記録・編集
  - **きっかけタグ**（ストレス・食後・コーヒー…）と**満足度**（満足／ふつう／惰性）
  - **在庫管理**（あと何本・約何日分・1箱買い足し・残り少のお知らせ）
  - ふりかえり（本数の推移・きっかけ・時間帯・満足度の傾向、コスト）
  - 前回からの経過時間、今日／今週／今月の本数、1日の目安本数（任意）

## アイコンの作り直し

```sh
convert -background none icons/icon.svg -resize 192x192 icons/icon-192.png
convert -background none icons/icon.svg -resize 512x512 icons/icon-512.png
```

## デプロイ（Cloudflare Pages）

1. このリポジトリを Cloudflare Pages に接続
2. ビルド設定：フレームワークなし／ビルドコマンド空欄／出力ディレクトリ `/`
3. カスタムドメイン（例：`ember.xdcyw.net`）を割り当て
